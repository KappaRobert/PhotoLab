Приложение для создания и управления заказами фотолаборатории.

Инструкция по запуску

    Сохраните код в файл app.py
    Создайте файл requirements.txt с зависимостями
    Установите зависимости:

pip install -r requirements.txt

    Запустите приложение:

python app.py

Функциональность приложения:

Для клиентов:

    Регистрация и авторизация
    Создание заказов с загрузкой файлов
    Просмотр статуса заказов
    Поиск по заказам
    Личный кабинет со статистикой

Для администраторов и сотрудников:

    Панель управления с общей статистикой
    Управление всеми заказами
    Изменение статусов заказов
    Управление услугами (добавление, редактирование)
    Просмотр информации о клиентах

Ключевые возможности:

    Автоматическая генерация номеров заказов
    Загрузка и хранение изображений
    Расчет стоимости и сроков выполнения
    Отслеживание статусов заказов
    Адаптивный интерфейс
    Система ролей пользователей

Тестовые данные:

    Администратор: admin / admin123
    Сотрудник: employee / emp123

Приложение готово к использованию и включает все необходимые функции для управления фотолабораторией!
